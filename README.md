# 🏥 One Life Medical Center (ASP.NET MVC)

A modern hospital website built using **ASP.NET Core MVC**, designed to provide an easy and user-friendly experience for patients to explore medical services, doctors, and book appointments online.

---

## 🌐 Live Demo  
👉 https://onelifemedicalcenter.runasp.net/

---

## 📌 Features

- 🩺 Doctors listing with images and details  
- 📅 Appointment booking page  
- 📰 Blog and medical articles  
- 👨‍⚕️ Team & testimonials section  
- 💲 Services and pricing pages  
- 🔍 Search functionality  
- 📱 Responsive design using Bootstrap  

---

## 🛠️ Technologies Used

- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- Razor Views  
- Bootstrap  
- HTML5 / CSS3 / JavaScript
